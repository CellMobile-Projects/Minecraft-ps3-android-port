# Minecraft-ps3-android-port
This is a port of Minecraft PlayStation 3 Edition to Android.
When you download the code as zip, unzip e zip inside the zip and then create a temporary world and then move the "Tutorial Level" folder to games/com.mojang/minecraftWorlds/ and then you can play the tutorial level.
If you feel like it, just download the zip file and not the code. Also I fixed the tutorial so it doesn't have random loot, but you can still get the old version of the tutorial world.
